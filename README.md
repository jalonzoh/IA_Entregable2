# IA_Entregable2
Proyecto Guayerd de análisis de ventas, ETL y uso de graficos.

# Tienda Aurelion : Optimización de Estrategias de Ventas y Fidelización de Clientes . 
En este Desafio se realizo ETL, análisis exploratorio, creación de gráficos que abarca cómo mejorar la forma en que vendes y cómo retienes a tus clientes, lo cual es fundamental para cualquier negocio.


## ⚡ **Introducción**

La empresa que ofrece servicios de venta alimentos y limpieza para mejorar su volumen de ventas, identificando las oportunidades principales.

## ⚡ **Objetivos**

### **Objetivo general**
  
Analisar e identificar estrategias para el incremento de ventas por categoria y sucursal

### **Objetivos específicos**

  * Crear una conexion dinamica usando la url.
  * Normalizar los datos del texto a atraves de la funciones en python.
  * Realizar el tratamiento de datos (limpieza, transformaciones, eliminación de duplicados).
  * Identificar relaciones entre variable objetivo de estudio con las variables categóricas y numéricas.
  * Representar gráficamente los datos.
